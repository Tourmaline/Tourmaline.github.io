## What is a copula?

A copula is a mathematical tool that helps us connect multiple variables together to understand how they depend on each other, without worrying about their individual distributions.

Each variable can have its own distribution (like normal, exponential, etc.). Computing CDF (cumulative distribution function) for each variable transforms them into uniform variables on [0, 1]. Now we have a set of uniform variables and this standardization allows us to focus on the **dependence structure** between the variables. 

Sklar's theorem states that we can write any joint distribution $F$ as a combination of its marginals $F_1, F_2, \ldots, F_n$ and a copula $C: [0, 1]^n \to [0, 1]$ that captures the dependence structure:

$$
F(x_1, x_2, \ldots, x_n) = C(F_1(x_1), F_2(x_2), \ldots, F_n(x_n)).
$$

A useful analogy is to think of a copula as a way to compare students from different schools with different grading systems. One school grades on a scale of 1-100, while another uses letter grades (A, B, C). To compare them fairly, one can convert all grades to percentiles to get each student's relative standing within their own school. Each school's transformed grades are then **uniformly distributed**. Now one can compare schools and students within these schools on a common scale, and analyze how students from different schools perform relative to each other. 

### Correlation vs. copula

Correlation matrix summarizes the **linear relationships** between variables, but it doesn't capture **nonlinear dependencies** or **tail dependencies** (how variables behave in extreme cases). This is a very important point to keep in mind! Two variables can have a correlation of zero but still be strongly dependent in a nonlinear way. For example, if $Y = X^2$ and $X$ is symmetrically distributed around zero, then the correlation between $X$ and $Y$ will be zero, even though $Y$ is completely determined by $X$.

For some joint distributions the dependence structure can be fully captured by the correlation matrix (e.g. multivariate normal), but for many others it cannot. Copulas allow us to model and analyze these more complex dependencies that go beyond linear correlation. 



## Sampling a multivariate normal with given covariance

Let $X_1$ and $X_2$ be normally distributed random variables with the same mean and variance, and covariance $c$. We want to generate a random vector:

$$
\mathbf{X} = \begin{bmatrix} X_1 \\ X_2 \end{bmatrix}
$$

such that
$\mathbf{X} \sim \mathcal{N}(\boldsymbol{\mu}, \Sigma)$, 

$$
\boldsymbol{\mu} = \begin{bmatrix} \mu \\ \mu \end{bmatrix} \quad \text{and} \quad \Sigma = \begin{bmatrix} \sigma^2 & c \\ c & \sigma^2 \end{bmatrix}.
$$

This means we want to sample from a bivariate normal distribution with a specific covariance structure.


We start by generating independent standard normal variables, and then apply a linear transformation to introduce the desired correlation. 

Let $Z_1, Z_2 \sim \mathcal{N}(0, 1)$ be two independent standard normal random variables, and let 

$$
\mathbf{Z} = \begin{bmatrix} Z_1 \\ Z_2 \end{bmatrix}.
$$

We are looking for a matrix $L$ such that $\mathbf{X} = \boldsymbol{\mu} + L\mathbf{Z}$.
This transformation should ensure that $\mathbf{X}$ has the desired covariance structure:

$$
\mathrm{Cov}(\mathbf{X}) = \mathrm{Cov}(L\mathbf{Z}) = L \cdot \mathrm{Cov}(\mathbf{Z}) \cdot L^\top = L \cdot I \cdot L^\top = LL^\top = \Sigma
$$

To find such a matrix $L$, we can use **Cholesky decomposition**, which factors a positive-definite matrix into the product of a lower triangular matrix and its transpose.




---

#### Cholesky of a 2×2 covariance matrix

Let's compute the Cholesky decomposition of:

$$
\Sigma = \begin{bmatrix} \sigma^2 & c \\ c & \sigma^2 \end{bmatrix}
$$

We want:

$$
L = \begin{bmatrix} a & 0 \\ b & d \end{bmatrix} \quad \text{such that} \quad LL^\top = \Sigma
$$

Compute $LL^\top$:

$$
LL^\top = \begin{bmatrix} a^2 & ab \\ ab & b^2 + d^2 \end{bmatrix}
$$

Set this equal to $\Sigma$:

$$
\begin{aligned}
a^2 &= \sigma^2 \\
ab &= c \\
b^2 + d^2 &= \sigma^2
\end{aligned}
$$

Solve:

- $a = \sqrt{\sigma^2} = \sigma$
- $b = \dfrac{c}{a} = \dfrac{c}{\sigma}$
- $d = \sqrt{\sigma^2 - b^2} = \sigma\sqrt{1 - \rho^2}$, where $\rho = \dfrac{c}{\sigma^2}$

So:

$$
L = \begin{bmatrix} \sigma & 0 \\ \dfrac{c}{\sigma} & \sigma\sqrt{1 - \left(\dfrac{c}{\sigma^2}\right)^2} \end{bmatrix}
$$

--- 

Now, given independent standard normals $Z_1, Z_2$, you compute:

$$
X_1 = \mu + \sigma Z_1
$$

$$
X_2 = \mu + \frac{c}{\sigma} Z_1 + \sigma\sqrt{1 - \left(\frac{c}{\sigma^2}\right)^2} Z_2
$$

This guarantees that $X_1$ and $X_2$ have the desired mean, variance, and covariance.


### Implementation

```python
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng()

mu = 0          # Mean
sigma2 = 1.0    # Variance
n = 1000        # Number of samples

sigma = np.sqrt(sigma2) # Standard deviation
c = 0.6  # Covariance

# Generate independent standard normals
z1 = rng.standard_normal(n)
z2 = rng.standard_normal(n)

# Transform to correlated normals
x1 = mu + sigma * z1
x2 = mu + c / sigma * z1 + sigma * np.sqrt(1 - (c / sigma**2)**2) * z2
```

## Generating correlated non-Gaussian variables with Gaussian copula

Now let $X_1$ and $X_2$ be non-Gaussian random variables with arbitrary marginal distributions $F_1$ and $F_2$, but we still want them to have a specific correlation $\rho$.

In the previous section we saw that generating correlated Gaussian variables can be done using:

$$
Z_2 = \rho Z_1 + \sqrt{1 - \rho^2} \cdot \epsilon, \quad Z_1, \epsilon \sim \mathcal{N}(0, 1)
$$




Apply the **standard normal CDF** $\Phi$ to each:

$$
U_1 = \Phi(Z_1), \quad U_2 = \Phi(Z_2)
$$

Now $U_1, U_2 \sim \mathrm{Uniform}(0, 1)$, but they are **correlated** due to the dependence between $Z_1$ and $Z_2$. The joint distribution of the pair $(U_1, U_2)$ is a Gaussian copula, implementing the desired correlation structure.

Now transform the uniforms into your desired marginals by applying the **inverse CDFs** (quantile functions) of your target distributions:

$$
X_1 = F_1^{-1}(U_1), \quad X_2 = F_2^{-1}(U_2)
$$

This procedure allows you to generate samples from a joint distribution with specified marginals and a Gaussian copula dependence structure. 



### Pseudocode (for Gaussian copula)

```
Input:
- n_samples: number of samples to generate
- rho: correlation coefficient between variables
- F1_inv, F2_inv: inverse CDFs of the marginal distributions

1: Define the correlation matrix
    Sigma = [[1, rho],
             [rho, 1]]

2: Generate samples from a bivariate normal distribution
    For i = 1 to n_samples:
        - Sample z = [z1, z2] from bivariate normal distribution with correlation Sigma
        (using Cholesky decomposition)

3: Convert normal samples to uniform
        - u1 = CDF_normal(z1)
        - u2 = CDF_normal(z2)

4: Apply inverse marginal CDFs to get final samples
        - x1 = F1_inv(u1)
        - x2 = F2_inv(u2)

Output:
- A list of (x1, x2) pairs representing samples from the joint distribution defined by the Gaussian copula and marginals
```



## t-Copula and tail dependence


The t-copula introduces symmetric tail dependence, meaning it increases the likelihood of both variables being extreme at the same time.

To sample from a joint distribution defined by a t-copula with correlation $\rho$ and degrees of freedom $\nu$, one can use a similar approach as with the Gaussian copula above, but instead of sampling from a multivariate normal, we sample from a multivariate t-distribution in step 2, and then apply the t-distribution CDF to get uniforms in step 3. 


## Visual comparison of Gaussian vs. t-copula

Here we compare joint distributions with Gaussian marginals, Gaussian copula vs t-copula with $\nu = 1$.


```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import t, norm

# Parameters
n = 50000
rho = 0.5
nu = 1  # degrees of freedom for t-copula

rng = np.random.default_rng()

# --- Gaussian Copula ---
# Generate independent standard normals
z1 = rng.standard_normal(n)
z2 = rng.standard_normal(n)
# Transform to correlated normals
z2 = rho * z1 + np.sqrt(1 - rho**2) * z2

x1_gauss = z1
x2_gauss = z2

# --- t-Copula ---
# Sample from a multivariate normal, then divide by a shared random
# scaling factor (introduces heavy tails and tail dependence)

# Generate independent standard normals
z1 = rng.standard_normal(n)
z2 = rng.standard_normal(n)
# Transform to correlated normals
z2 = rho * z1 + np.sqrt(1 - rho**2) * z2

# Sample shared scaling factor
w = rng.chisquare(nu, size=n)
s = np.sqrt(nu / w)

# Scale the normal samples
t1 = z1 * s
t2 = z2 * s

# Convert to uniforms
u1 = t.cdf(t1, df=nu)
u2 = t.cdf(t2, df=nu)

# Apply inverse marginals (standard normal)
x1_t = norm.ppf(u1)
x2_t = norm.ppf(u2)

# --- Plotting ---
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

ax1.scatter(x1_gauss, x2_gauss, alpha=0.5, color="blue", s=5)
ax1.set_title("Gaussian Copula")
ax1.set_xlabel("X1")
ax1.set_ylabel("X2")
ax1.grid()

ax2.scatter(x1_t, x2_t, alpha=0.5, color="red", s=5)
ax2.set_title(f"t-Copula (ν = {nu})")
ax2.set_xlabel("X1")
ax2.set_ylabel("X2")
ax2.grid()

plt.tight_layout()
```

This example shows how copula captures the dependence structure between variables. Both cases have same marginals and correlation, but the t-copula introduces tail dependence, which is not present in the Gaussian copula.
